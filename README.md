# AniBot

A Discord bot built with Discord.js and TypeScript.

## Features

- Command handling
- Event handling
- Cooldown management
- ...

## Setup

1. Clone the repo:

   ```bash
   git clone https://github.com/your-username/anibot.git
   cd anibot
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Add an `.env` file with the bot token:
   ```ini
   TOKEN= token_here
   ```
4. Compile and run:
   ```bash
   npm run build
   node dist/index.js
   ```
